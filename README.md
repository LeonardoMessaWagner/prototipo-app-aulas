# Protótipo — App de Aulas Bilíngues

Protótipo navegável (demonstração) do aplicativo para uma escola de aulas
bilíngues de inglês, com foco na comunicação oral.

**Perfis demonstrados:** Aluno · Teacher · Admin (dona da escola).

**Funcionalidades no protótipo:**
- Homework/Tema — criar (com escolha de formato de resposta), responder (texto/foto/áudio), corrigir (comentário + estrelas de 1 a 5)
- Conteúdos — biblioteca de filmes, séries, vídeos e músicas por nível
- Agenda — aulas em domicílio com lembretes aos pais
- Avisos — comunicação com os pais
- Painel da Admin — visão da escola inteira

> Este é um protótipo visual para apresentação. Dados são fictícios e as telas
> simulam a navegação; não há backend real.

## Como ver

Abra o `index.html` no navegador, ou acesse a versão publicada no GitHub Pages.
